# 🇷🇺 Русификатор Hytale | Полная локализация

Самый полный и актуальный русификатор для Hytale.  
Переведено всё: от главного меню до описаний предметов и настроек персонажа.

---

## 📥 Установка

Выберите вашу операционную систему:

<details>
<summary><b>🪟 Windows (нажмите, чтобы развернуть)</b></summary>
<br>

### Установка
1. **[Скачайте русификатор](https://github.com/Synth3s/Hytale-Russian/releases/latest/download/Russian_Localization.zip)**.
2. Откройте папку игры:  
   В лаунчере нажмите ⚙️ → **Open Directory**.
3. Скопируйте содержимое архива в корневую папку игры.
4. Подтвердите замену файлов.

---

### ⚠️ ВАЖНО (обновления игры)

Лаунчер проверяет файлы при каждом обновлении.  
Перед обновлением игры:

1. Скачайте **[оригинальный шрифт](https://github.com/Synth3s/Hytale-Russian/releases/download/v1.0/Lexend-Bold.zip)**.
2. Скопируйте его по пути:  
   `install\release\package\game\latest\Client\Data\Shared\Fonts`
3. Запустите обновление игры в лаунчере.
4. После обновления снова установите русификатор.

---

### 🚫 Assets.zip

**Assets.zip НЕ ТРОГАТЬ.**  
Лаунчер проверяет этот файл при обновлении - его изменение приводит к ошибкам. Мы пока не нашли решение для установки полного русификатора на клиент.
</details>

---

<details>
<summary><b>🍎 macOS (нажмите, чтобы развернуть)</b></summary>
<br>

### Установка
1. **[Скачайте русификатор](https://github.com/Synth3s/Hytale-Russian/releases/latest/download/Russian_Localization.zip)**.
2. В папке `Client` нажмите правой кнопкой мыши на приложение **Hytale** → **Показать содержимое пакета**.
3. Перейдите по пути:  
   `Contents/Resources/Data/Shared/`
4. Скопируйте файлы:
   - из папки `Fonts` → в папку `Fonts` игры  
   - из папки `Language/ru-RU` → в папку `Language/ru-RU` игры

---

### 💡 Совет
При копировании удерживайте клавишу **Option (Alt)** и выбирайте **«Объединить» (Merge)**, чтобы не удалить оригинальные файлы.

---

### ⚠️ ВАЖНО (обновления игры)

Перед обновлением игры:

1. Скачайте **[оригинальный шрифт](https://github.com/Synth3s/Hytale-Russian/releases/download/v1.0/Lexend-Bold.zip)**.
2. Скопируйте его в папку:  
   `Hytale.app/Contents/Resources/Data/Shared/Fonts`
3. Запустите обновление в лаунчере.
4. После обновления снова установите русификатор.

---

### 🚫 Assets.zip

**Assets.zip НЕ ТРОГАТЬ.**  
Лаунчер проверяет этот файл при обновлении - его изменение вызывает ошибки.

На сервере русификатор по-прежнему можно устанавливать.

</details>

## 💡 Важная информация

> [!IMPORTANT]
> **Для игроков:**
> 
> * На сервере **`playhytale.su`** русификатор уже установлен.  
>   Достаточно установить перевод только на клиент - остальное подгрузится автоматически.

---

> [!TIP]
> **Для владельцев серверов:**
> 
> Чтобы игроки видели перевод предметов и интерфейса:
> 1. Распакуйте `Assets.zip` на сервере.
> 2. В корневую директорию сервера распакуйте **[серверный русификатор](https://github.com/Synth3s/Hytale-Russian/releases/download/v1.0/Assets.zip)**.
> 3. Не забудьте в параметрах запуска сервера заменить путь к ресурсами с `Assets.zip` на путь к папке `Assets`.
